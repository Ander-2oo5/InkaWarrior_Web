## 📌 Sobre el proyecto
**InkaWarrior Web** es una página web desarrollada con un enfoque moderno, pensada para presentar de forma atractiva el concepto/identidad de **InkaWarrior**.  
El objetivo es brindar una experiencia visual clara, rápida y adaptable a cualquier dispositivo. 📱💻

---

## ✅ ¿Qué encontrarás aquí?
- 🖼️ Interfaz **visual y llamativa**
- 📱 Diseño **responsive** (móvil y escritorio)
- ⚡ Navegación fluida y estructura ordenada
- 🟦 Desarrollo principalmente con **TypeScript**
- 🎨 Estilos con **CSS**
- 🧱 Base estructural en **HTML**

## 🚀 Cómo ejecutar el proyecto

### ✅ Requisitos
- Tener instalado **Node.js (LTS)**
- Un gestor de paquetes: **npm** (recomendado) / **pnpm** / **yarn**

### 1️⃣ Clonar el repositorio
```bash
git clone https://github.com/Ander-2oo5/InkaWarrior_Web.git
cd InkaWarrior_Web
```

### 2️⃣ Instalar dependencias
```bash
npm install
```

### 3️⃣ Ejecutar en modo desarrollo
```bash
npm run dev
```

📌 Luego abre la URL que te muestre la terminal (normalmente):
- `http://localhost:5173` (Vite)
- o `http://localhost:3000`

### 4️⃣ Compilar para producción
```bash
npm run build
```

### 5️⃣ Ejecutar el proyecto compilado (si aplica)
```bash
npm run preview
# o
npm run start
```
